QGIS Tutorial

A Tutorial for QGIS - Currently Written for QGIS v2.4

Download this project as a .zip file Download this project as a tar.gz file

Items covered in this tutorial include importing spatial data, dealing with projections, basic raster and vector manipulations, using some plugins, and creating a map.


This material was produced for the 2014 Open Source for Open Science Workshop, as part of the Ecology and Evolutionary Biology Program at Texas A&M University.

  • All of sample files in a .zip file
