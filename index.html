QGIS Tutorial

A GIS Tutorial using QGIS - Currently Written for QGIS v2.12.2

Download this project as a .zip file Download this project as a tar.gz file

Items covered in this tutorial include importing spatial data, dealing with projections, basic raster and vector manipulations, using some plugins, and creating a map.



Use the links below to access the tutorial and data files:

  • Tutorial as a PDF
  • Sample data in a .zip file

Alternatively, download all materials (including archived versions) as a .zip or Tarball file using one of the icons above, or go to the main GitHub repository.

If you are intersted in an earlier version, for 2.6.1, click here, with the original sample data avaialble from this link.
